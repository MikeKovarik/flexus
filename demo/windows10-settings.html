Windows Defender

Windows Defender protects your computer against viruses, spyware and other malicious software. Open Windows Defender to use it.

Real-time protection

This helps find and stop malware from installing or urnning on your PC. You can turn this off temporarily, but if it's off for a while we'll turn it back on automatically.

Cloud-based Protection

Get Real-time protection when Windows Defender sends info to Microsoft about potential security threats. This feature works best with Automatic sample submission enabled.

Privacy Statement

Automatic sample submission

Allow windows Defender to send samples of suspicious files to Microsoft, to help improve malware detection. Turn this off to be prompted before sending samples to Microsoft.

Privacy Statement

Exclusions

Windows Defender won't scan excluded files, making your PC more vulnerable to mallware.

Add an exclusion

Enhanced notifications

Windows Defender sends notifications to help nesure you are informed about the health of your PC. Even if this options is turned off, you'll still get critical notifications for issues that need immediate attention.

Privacy Statement

Windows Defender Offline

Some malicious software can be particularly difficult to remove from your PC. Windows Defender Offline can help find and remove them using up-to-date threat definitions. This will restart your pc and wil take about 15 minutes.
