Flexus tries to mix and match icons from both icon sets. Both Segoe and Material have currently over thousand icons, but not all of them have similar counterpart in the other icon library.

Following list contains about 400 icons (some of them are aliasm like 'add', 'plus' and 'create') that are avaiable in both icon sets and are safe to use.

Usage

Universal [icon] attribute can be used on most elements where ::before pseudo element is not already in use:

<button icon="insert-name-here"></button>

<div fx-item icon="insert-name-here"></div>

Otherwise <i> element can be used:

<i icon="insert-name-here"></i>
